# well-beeing

## Project setup

```
dnf install git
dnf install npm
mkdir /var/www
cd /var/www

git clone https://github.com/MicroHealthLLC/well-beeing.git

npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
Default port will be URL:8080

### Compiles and minifies for production
```
npm run build
```
Default port will be URL:8080

### Lints and fixes files
```
npm run lint
```
